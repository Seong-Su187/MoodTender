건강 & 분석

Health Dashboard

스마트폰 라이프로그(걸음수·수면·스크린타임)를 분석해 "왜 요즘 이상한지"를 데이터로 설명. 바텐더 어투로 즉시 실천 가능한 행동을 제안.

학술적 근거

근거연구
수면 부족 → 부정 감정 증폭Harvey et al., 2011 / Walker, 2017
신체활동 감소 → 우울·무기력Blumenthal et al., 2007
야간 스크린타임 → 과각성·불안Saeb et al., 2015
보복성 철야 증후군Kroese et al., 2014
소셜 고립 → 외로움Holt-Lunstad et al., 2015
개인 내 변동성 기반 분석 (Baseline vs Delta)Nesselroade, 1991
RAG — 검증된 심리·행동요법 지식 주입activity_knowledge DB

월간 감정 분석

Monthly Analysis

매일 대화로 쌓인 감정 기록을 월 단위로 돌아보며 "이번 달 나는 어떤 상태였나"를 흐름으로 파악. 심리학적 해석을 바텐더 언어로 제공.
